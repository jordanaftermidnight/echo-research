# Security Policy

## Repository Scope

This repository contains **documentation and conceptual materials only**.

No implementation code, model weights, session data, detector logic,
probing algorithms, or private configuration is present or will ever be
committed to this repository.

## Reporting Issues

If you believe sensitive information has been inadvertently disclosed,
please open a private security advisory rather than a public issue.

## For Collaborators and Contributors

- Do **not** submit PRs containing implementation details
- Do **not** attempt to reconstruct the detection methodology from the
  whitepaper for competitive or commercial purposes
- Academic and research use of the documented concepts is welcome
  under the terms of the project licence

## Intellectual Property

The ECHO system, detection suite, probing methodology, and all
associated implementation materials are proprietary. The whitepaper
and architecture documentation are released for research communication
purposes only.
