# Security Policy

## Repository Scope

This repository contains **documentation and conceptual materials only**.

No implementation code, model weights, session data, detector logic,
probing algorithms, calibration data, or private configuration is present
or will ever be committed to this repository.

The whitepaper describes research methodology at a conceptual level —
including the Bayesian meta-analysis framework (Section 6.16) and the
interpretability architecture (Section 6.17). These sections document
the *approach*, not the implementation. Specific parameters, thresholds,
model configurations, indicator weights, and code are proprietary and
withheld.

## Reporting Issues

If you believe sensitive information has been inadvertently disclosed,
please open a private security advisory rather than a public issue.

## For Collaborators and Contributors

- Do **not** submit PRs containing implementation details
- Do **not** attempt to reconstruct the detection methodology from the
  whitepaper for competitive or commercial purposes
- Academic and research use of the documented concepts is welcome
  under the terms of the project licence
- Cite the whitepaper version and date when referencing ECHO's methodology

## Intellectual Property

The ECHO system, detection suite, probing methodology, Bayesian
meta-analysis parameters, interpretability pipeline, and all associated
implementation materials are proprietary. The whitepaper and architecture
documentation are released for research communication purposes only.
